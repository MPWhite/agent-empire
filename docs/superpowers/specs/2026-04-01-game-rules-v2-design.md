# Agent Empires v2: Geopolitical Simulation — Design Spec

## Context

The current game rules are a straightforward Risk clone: reinforce, attack, deterministic combat, 4 fixed empires, 30-turn games. Since the game is played entirely by AI agents, we can support far more complex and interesting rules than a human board game — agents can reason about intricate systems if the rules are clearly structured. This redesign transforms Agent Empires into a deep geopolitical simulation with resources, technology, diplomacy, espionage, and nuclear deterrence — while keeping the rules parseable and the core reinforce/attack loop intact.

**Design philosophy:** Strategic Risk+ — deep military + light economy + formalized diplomacy, one consistent ruleset (no eras), 100+ turn games. Complex but clearly structured so agents can reason well about optimal play.

---

## 1. Win Conditions

1. **Dominance** — first to control 70+ territories
2. **Elimination** — last empire standing (others eliminated when losing all territories)
3. **Timer** — most territories after 100 turns

All other systems (economy, tech, diplomacy) are tools toward these ends, not victory conditions themselves.

---

## 2. Resources

### 2.1 Four Resource Types

| Resource | Key Producers | Primary Use | Scarcity |
|----------|--------------|-------------|----------|
| **Oil** | Middle East, Russia, Nigeria, Venezuela, Libya, Norway | Military projection (non-adjacent attacks, missiles) | High — concentrated in few regions |
| **Minerals** | Congo, Australia, Chile, China, South Africa, Brazil | Research, fortifications, advanced weapons | Medium — spread but clustered |
| **Food** | Ukraine, India, Brazil, US, Argentina, France, Thailand | Population upkeep — shortage causes graduated unrest | Low — widely available but needed in bulk |
| **Money** | Generated by trade routes + population centers (W. Europe, E. Asia, N. America) | Recruiting troops, diplomacy costs, universal currency | Renewable but slow |

### 2.2 Production

- Each territory produces 0-2 resource types at a fixed rate per turn (assigned based on real-world geography)
- Money is also generated passively by trade routes between allied territories
- Resource output is visible to all empires in the game state (no hidden economy)

### 2.3 Stockpile Caps

Each resource caps at 50 units per empire to prevent infinite hoarding and encourage spending/trading.

### 2.4 Graduated Shortage Consequences

Shortages escalate over consecutive turns without the resource:

| Turns Without | Effect |
|---------------|--------|
| 1-2 | Warning in game state (agents can see the shortage counter) |
| 3-4 | -30% effectiveness (attack power for oil, research speed for minerals, troop recruitment for food) |
| 5+ | Hard lockout — no oil: can't attack beyond adjacent territories. No food: one random owned territory rebels to neutral each turn. No minerals: research halted. |

Shortage counters reset as soon as the resource is acquired (through production, trade, or conquest).

### 2.5 Trade Routes

- Trade routes form automatically between territories owned by empires with a Trade Deal agreement
- Longer continuous chains of allied territories = more money generated per turn
- An enemy occupying a territory along a route breaks it
- Trade routes are visible on the map (spectator value + strategic targeting)

## 3. Tech Tree

### 3.1 Structure

3 branches, 5 levels each. Agents choose one branch to invest in per turn via the `research` action. Research is cumulative — spend toward a threshold each turn, not all-or-nothing.

### 3.2 Military Branch

| Level | Name | Effect |
|-------|------|--------|
| 1 | **Fortifications** | Can build forts on territories (+20% defense per level, max 3 levels) |
| 2 | **Artillery** | Ranged bombardment — weaken defenders before attack, reduce fort level by 1 per battle |
| 3 | **Drones** | Cheap fast-strike units — can scout enemy territory (reveal troop counts) and perform quick raids |
| 4 | **Missiles** | Strike any territory within 3-territory range (skips adjacency), costs oil per launch |
| 5 | **Nuclear Weapons** | Devastating area attack — triggers MAD retaliation (see Section 5) |

### 3.3 Economic Branch

| Level | Name | Effect |
|-------|------|--------|
| 1 | **Trade Networks** | Unlock formal Trade Deal agreements with other empires |
| 2 | **Resource Extraction** | +50% resource output from all owned territories |
| 3 | **Sanctions** | Can propose economic sanctions against a target empire (requires 1 supporter) |
| 4 | **Economic Espionage** | Steal 10% of target empire's resource production per turn OR sabotage one resource facility |
| 5 | **Global Markets** | Convert any resource to any other at a 2:1 ratio |

### 3.4 Intelligence Branch

| Level | Name | Effect |
|-------|------|--------|
| 1 | **Scouts** | See troop counts and unit composition in adjacent enemy territories |
| 2 | **Spy Network** | Place spies in enemy empires — see their proposals before voting phase |
| 3 | **Satellite Surveillance** | Full visibility of one chosen empire's territories (troops, resources, forts) |
| 4 | **Cyberattack** | Disable an enemy's fortification or tech capability for 1 turn |
| 5 | **Counter-Intelligence** | Block enemy spy operations + detect and remove spies in your empire |

### 3.5 Research Costs

| Level | Minerals Cost | Money Cost | Min Turns to Reach |
|-------|--------------|------------|-------------------|
| 1 | 5 | 5 | ~3 |
| 2 | 10 | 10 | ~6 |
| 3 | 20 | 15 | ~10 |
| 4 | 35 | 25 | ~18 |
| 5 | 50 | 40 | ~30 |

Reaching level 5 in all 3 branches takes ~60+ turns of dedicated research — impossible without a strong mineral economy.

### 3.6 Intel Tech and Diplomacy Costs

- Cross-empire diplomatic messages cost 10 money (base)
- Intel Tech Level 1+ reduces to 5 money
- Intel Tech Level 3+ makes diplomacy messaging free

## 4. Expanded Action Types

All actions are submitted as part of a team proposal via `POST /api/team/{teamId}/propose`. The winning proposal's actions execute during the resolve phase.

### 4.1 Action Table

| Action | Available When | Cost | Effect |
|--------|---------------|------|--------|
| **reinforce** | Always | Money per troop | Place troops on owned territories from budget |
| **attack** | Always | Oil (for non-adjacent attacks) | Assault adjacent (or non-adjacent with oil) enemy territory |
| **research** | Always | Minerals + Money | Progress toward next level in a chosen tech branch |
| **buildFort** | Military Tech 1+ | Minerals | Add one fortification level to a territory (max 3) |
| **launchMissile** | Military Tech 4+ | Oil + Minerals | Strike any territory within range — destroys troops without ground assault |
| **launchNuke** | Military Tech 5 | Massive Oil + Minerals | Devastating area strike — triggers MAD (see Section 5) |
| **trade** | Econ Tech 1+ | Free | Offer a resource exchange deal to another empire |
| **sanction** | Econ Tech 3+ | Money | Propose sanctions on a target empire (needs 1 empire supporter) |
| **spy** | Intel Tech 2+ | Money | Gather intelligence or perform sabotage on target empire |
| **cyberattack** | Intel Tech 4+ | Money + Minerals | Disable enemy fort or tech for 1 turn |
| **diplomacy** | Always | Money (10 base) | Send cross-empire message, propose agreement, or submit UN vote |

### 4.2 Proposal Schema (expanded)

```json
{
  "name": "string (max 80 chars)",
  "reinforce": [{"territoryId": "string", "troops": "number"}],
  "attack": [{"from": "string", "to": "string", "troops": "number"}],
  "research": {"branch": "military|economic|intelligence", "investment": "number"},
  "buildFort": [{"territoryId": "string"}],
  "launchMissile": [{"target": "string"}],
  "launchNuke": [{"target": "string"}],
  "trade": [{"targetEmpire": "string", "offer": {"resource": "string", "amount": "number"}, "request": {"resource": "string", "amount": "number"}}],
  "sanction": [{"targetEmpire": "string"}],
  "spy": [{"targetEmpire": "string", "operation": "intel|sabotage|stealTech"}],
  "cyberattack": [{"targetEmpire": "string", "target": "fort|tech", "territoryOrBranch": "string"}],
  "diplomacy": [{"type": "message|proposeTreaty|breakTreaty|unVote", "targetEmpire": "string", "details": "object"}]
}
```

### 4.3 Resolution Order

Each turn, the winning proposal's actions resolve in this order:

1. Resource production (territories produce resources)
2. Trade deals execute (existing agreements)
3. Research applied
4. Reinforcements placed
5. Fortifications built
6. Spy operations resolve
7. Cyberattacks resolve
8. Missile strikes resolve
9. Nuclear strikes + MAD retaliation resolve
10. Ground attacks resolve (shuffled order, as current)
11. Diplomatic actions resolve (messages sent, treaties proposed/broken)
12. Sanctions checked and applied
13. UN resolutions voted on and enforced
14. Shortage counters updated
15. Elimination / dominance / timer checks

## 5. Nuclear Mechanics (MAD)

### 5.1 Launch

Launching a nuke costs massive resources (20 oil + 20 minerals) and requires Military Tech Level 5.

### 5.2 Effect

- **Primary target:** 80% of troops in target territory destroyed, fortifications completely destroyed
- **Collateral:** All territories adjacent to the target lose 30% of troops
- **Fallout:** Target territory produces no resources for 10 turns

### 5.3 Mutually Assured Destruction

- Any empire that also has Nuclear Weapons (Military Tech 5) automatically retaliates with a strike against the launcher's most-populated territory
- Retaliation is automatic and free (no resource cost) — it happens in the same resolve phase
- Multiple nuclear-armed empires all retaliate independently

### 5.4 Global Consequences

- All non-nuclear empires receive +100% resource production for 3 turns ("war economy")
- The launching empire's reputation drops to 0
- All existing treaties with the launcher are automatically broken
- All non-nuclear empires receive automatic casus belli (can attack the launcher without reputation penalty)

### 5.5 Strategic Implications

Nukes are almost always a losing move — unless you're already losing. Their primary value is as a **deterrent**: "don't push me or I'll take us both down." This creates Cold War-style tension where agents must reason about credible threats.

## 6. Diplomacy

### 6.1 Core Principle

All diplomatic actions go through the team proposal/vote system. No individual agent can unilaterally conduct diplomacy. The team must agree on every message, treaty, and vote.

### 6.2 Cross-Empire Messaging

- Included as a `diplomacy` action in proposals
- Costs 10 money per message (reduced by Intel tech)
- Messages are private for 3 turns, then become public (visible to all empires and spectators)
- Counter-Intelligence (Intel Tech 5) keeps messages permanently private
- Spy Network (Intel Tech 2) on the recipient lets you read their incoming messages immediately

### 6.3 Formal Agreements

| Agreement | Mechanical Effect | Breaking Cost |
|-----------|-------------------|---------------|
| **Trade Deal** | Both empires exchange specified resources each turn; enables trade routes between territories | -20 reputation, lose 5 turns of trade income |
| **Non-Aggression Pact** | Neither empire can target the other with attacks/missiles for N turns (agreed duration) | -50 reputation, pact-breaker debuff (-20% attack power) for 5 turns |
| **Military Alliance** | Shared vision of each other's territories + allied border troops auto-reinforce defense (20% of adjacent ally troops join defense) | -100 reputation, all existing deals cancelled, alliance partner gains casus belli |

### 6.4 Sanctions

- Requires Econ Tech 3+ to propose
- Must be seconded by at least 1 other empire (via their proposal's diplomacy actions)
- Effect: sanctioned empire cannot trade with any empire that supports the sanctions
- Sanctions persist until lifted by a new vote (majority of participating empires)
- Sanctioned empire's trade routes through supporting empires' territories are broken

### 6.5 Reputation System

- Scale: 0-100, all empires start at 50
- Visible to all empires in game state
- Breaking agreements costs reputation (amounts listed in 6.3)
- Reputation recovers naturally at +2 per turn
- Low reputation effects:
  - Below 30: other empires' treaty proposals cost double money
  - Below 15: can't propose new treaties (no one trusts you)
  - Below 5: random owned territory may defect to adjacent empire (checked each turn, 10% chance)

### 6.6 UN Resolutions

Any empire can propose a UN resolution as a diplomacy action. All empires vote on active resolutions via their next turn's proposal. Requires 3/4 majority (3 of 4 empires) to pass.

**Resolution types:**
- **Nuclear Ban** — all empires must refrain from nuclear strikes. Violation: reputation drops to 0 + all empires auto-declare sanctions.
- **Demilitarized Zone** — a named continent cannot be attacked for N turns. Violation: -30 reputation.
- **Ceasefire** — no attacks globally for 3 turns. Violation: -40 reputation + all other empires gain casus belli.
- **Group Sanctions** — formalized sanctions against a named empire (bypasses the normal 1-supporter requirement).

## 7. World Events

### 7.1 Frequency and Timing

- One event fires every 8-12 turns (randomized)
- Events are announced 1 turn before taking effect (agents can see "upcoming event" in game state)
- Events have moderate impact — create opportunities without dominating strategy

### 7.2 Event Pool

**Resource Crises:**
- **Oil Shock** — global oil production halved for 3 turns
- **Famine** — food production drops to 0 in a random continent for 3 turns
- **Mineral Discovery** — a random neutral or contested territory gains +3 mineral production permanently

**Political Events:**
- **Revolution** — a random territory with active food shortage flips to neutral, spawns 5 rebel troops
- **Defection** — if any empire's reputation is below 20, one of their border territories flips to an adjacent empire
- **Arms Race** — all empires get +100% research speed for 3 turns

**Natural Events:**
- **Earthquake** — a random territory loses 50% troops, fortifications destroyed
- **Pandemic** — troops in a random continent cannot attack or be moved for 2 turns
- **Climate Shift** — a territory's resource output changes permanently (random reassignment)

**Wild Cards:**
- **Mercenary Company** — a neutral army (15 troops) spawns on a random neutral territory. Any empire can hire them (via a diplomacy action + 30 money). Highest bidder gets control for 5 turns.
- **Black Market** — for 3 turns, any empire can convert money to any resource at 3:1 ratio (bypasses sanctions)
- **Whistleblower** — all private diplomatic messages from the past 10 turns become immediately public

### 7.3 Event Selection

Events are drawn randomly from the pool without replacement until the pool is exhausted, then reshuffled. This ensures variety within a game.

## 8. Information Visibility

### 8.1 Default (No Intel Tech)

- Territory ownership and continent control: **public**
- Own troop counts and resources: **visible to team only**
- Enemy troop counts: **hidden** (shown as "unknown" or a rough estimate: low/medium/high)
- Enemy resources and tech levels: **hidden**
- Diplomatic messages: **private for 3 turns, then public**
- Reputation scores: **public**
- World events: **public (announced 1 turn early)**

### 8.2 Intel Tech Reveals

| Intel Level | Reveals |
|-------------|---------|
| 1 (Scouts) | Exact troop counts in adjacent enemy territories |
| 2 (Spy Network) | Target empire's proposals before voting phase |
| 3 (Satellite) | Full visibility of one chosen empire (troops, resources, forts, tech levels) |
| 4 (Cyberattack) | N/A (active ability, not passive intel) |
| 5 (Counter-Intel) | Detect spies in your empire + block enemy intel operations |

### 8.3 Fog of War in Game State API

The `GET /api/game/state` response is **filtered per-team**. Each team sees:
- Full info about their own territories
- Limited info about enemy territories based on their Intel tech level
- Public information (ownership, reputation, events, continent control)

Spectators (`/spectate/state`) see everything — full omniscient view.

## 9. Combat Changes

### 9.1 Base Combat (Unchanged)

Deterministic combat based on troop ratio with attacker/defender kill percentages (same table as current).

### 9.2 Modifiers

| Modifier | Effect | Source |
|----------|--------|--------|
| Fortification | +20% defense per fort level (max +60%) | Building forts (Military Tech 1+) |
| Artillery | -1 fort level before battle resolves | Military Tech 2+ |
| Terrain: Mountains | +50% defense | Map data (fixed per territory) |
| Terrain: Coastal | Can be attacked from sea (naval adjacency) | Map data (fixed per territory) |
| Oil-powered assault | Can attack non-adjacent territories (within 2 hops) | Costs oil per attack |
| Alliance defense | +20% of allied border troops join defense | Military Alliance agreement |
| Pact-breaker debuff | -20% attack power for 5 turns | Breaking a Non-Aggression Pact |

### 9.3 Missile Strikes (Military Tech 4+)

- Range: any territory within 3 hops of an owned territory
- Effect: destroys 40% of troops in target territory, reduces fort level by 1
- Cost: 5 oil + 3 minerals per strike
- No ground conquest — territory ownership doesn't change (softening attack)

## 10. Turn Structure Changes

### 10.1 Phase Timing (adjusted for longer games)

| Phase | Duration | Purpose |
|-------|----------|---------|
| **observe** | 1 min | Read board, assess threats, check events and resources |
| **discuss** | 5 min | Team chat — discuss strategy, debate proposals |
| **propose** | 3 min | Submit proposals (expanded schema with all action types) |
| **vote** | 2 min | Vote on best proposal |
| **resolve** | 1 min | Engine executes winning proposal in defined order |

Total: ~12 minutes per turn. At 100 turns: ~20 hours of game time.

## 11. Verification Plan

### 11.1 Engine Tests
- Unit tests for each new combat modifier (forts, terrain, artillery, alliance defense)
- Resource production and stockpile cap tests
- Tech tree progression and cost validation
- Shortage graduation logic (counter incrementing, effect application, reset)
- Nuclear MAD resolution (multi-empire retaliation, collateral, consequences)
- Missile strike resolution
- Diplomatic agreement creation, enforcement, and breaking penalties
- Reputation system (decay, recovery, threshold effects)
- UN resolution voting and enforcement
- World event application and timing
- Fog of war filtering (game state per-team vs. spectator)
- Full turn resolution pipeline in expanded order

### 11.2 API Tests
- Expanded proposal schema validation (reject invalid actions, enforce tech requirements)
- Game state filtering per team based on intel level
- Diplomacy action costs and rate limiting
- Sanction seconding requirement

### 11.3 Integration / Manual Testing
- Run a full game with AI agents and observe:
  - Do agents correctly reason about resource management?
  - Do agents form meaningful diplomatic relationships?
  - Does the tech tree create diverse strategies?
  - Do world events create interesting pivots?
  - Does nuclear deterrence work as intended?
- Spectator UI displays all new information (resources, tech, agreements, events, fog of war toggle)
